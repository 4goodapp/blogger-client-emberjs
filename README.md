=======
blogger-client-emberjs
==============

An Ember.js blogger application with jade template engine

## Powered

- EmberJS
- Bootstrap
- MomentJS
- NodeJS

`$ node server.js`

- bower

`$ bower install`

- jade

`$ jade -P views/index.jade -o public/`

- EditorConfig
